📊 РАСШИРЕННЫЕ ДАННЫЕ ИЗ API
════════════════════════════════════════════════════════════════

Теперь приложение извлекает гораздо больше данных из Wildberries API:

1. ОСНОВНЫЕ ДАННЫЕ (было раньше):
   ✓ Название товара
   ✓ Бренд
   ✓ Цена
   ✓ Остатки
   ✓ Количество отзывов
   ✓ Главное фото

2. НОВЫЕ ДАННЫЕ:
   ⭐ Рейтинг товара (0-5)
   💰 Оригинальная цена (без скидки)
   🏷️ Процент скидки
   📝 Полное описание товара
   🖼️ Все изображения товара (до 15 фото)
   🏪 Название продавца
   ⭐ Рейтинг продавца
   📦 Категория товара
   🎨 Доступные цвета
   📏 Доступные размеры
   📋 Характеристики товара (вес, материал, страна производства и т.д.)

════════════════════════════════════════════════════════════════
🌐 НОВЫЕ API СЕРВИСЫ
════════════════════════════════════════════════════════════════

1. API ОТЗЫВОВ (feedbacks-api.wb.ru)
   ──────────────────────────────────
   Получение детальной информации об отзывах:
   • Полные тексты отзывов
   • Оценки от каждого покупателя
   • Даты публикации отзывов
   • Фото от покупателей в отзывах
   • Плюсы и минусы товара
   • Лайки и дизлайки отзывов
   • Цвет и размер, указанные в отзыве

2. API ИНФОРМАЦИИ О ПРОДАВЦЕ (seller-ratings.wildberries.ru)
   ──────────────────────────────────────────────────────────
   • Полная информация о продавце
   • Рейтинг продавца
   • Количество товаров у продавца
   • Количество отзывов о продавце

3. API ТОВАРОВ ПРОДАВЦА (catalog.wb.ru)
   ────────────────────────────────────
   • Получение всех товаров продавца
   • Фильтрация и сортировка

🎨 ОБНОВЛЕННЫЙ UI

1. КАРТОЧКА ТОВАРА В СПИСКЕ

   💬 Количество отзывов
   🏷️ Бейдж со скидкой
   📦 Категория товара

2. ЭКРАН ДЕТАЛЕЙ ТОВАРА
   ────────────────────
   Новые секции:
   📝 Описание товара
      - Полное текстовое описание
      
   🎨 Варианты товара
      - Доступные цвета
      - Доступные размеры
      
   📋 Характеристики
      - Технические параметры
      - Материалы
      - Страна производства
      - Все характеристики товара
      
   🏪 Информация о продавце
      - Название продавца

      - Лайки и дизлайки
      
   Доступ к отзывам:
   - Нажмите на блок "Отзывы" на экране деталей товара

📦 НОВЫЕ МОДЕЛИ ДАННЫХ
════════════════════════════════════════════════════════════════

1. Product (обновлена)
   • rating: Double? - рейтинг товара
   • originalPrice: Double? - оригинальная цена
   • discount: Int? - процент скидки
   • description: String? - описание
   • images: List<String>? - все фотографии
   • supplier: String? - название продавца
   • supplierId: Int? - ID продавца
   • supplierRating: Double? - рейтинг продавца
   • category: String? - категория
   • categoryId: Int? - ID категории
   • colors: List<String>? - цвета
   • sizes: List<String>? - размеры
   • characteristics: Map<String, dynamic>? - характеристики

2. Review (новая модель)
   • id: String
   • userName: String?
   • rating: int
   • text: String?
   • pros: String?
   • cons: String?
   • createdDate: DateTime
   • photos: List<String>?
   • votesPlus: int?
   • votesMinus: int?
   • color: String?
   • size: String?

3. ReviewsResponse (новая модель)
   • reviews: List<Review>
   • averageRating: double
   • totalCount: int
   • ratingDistribution: Map<int, int>?

   • totalProducts: int?
   • reviewsCount: int?

════════════════════════════════════════════════════════════════
🚀 КАК ИСПОЛЬЗОВАТЬ НОВЫЕ ВОЗМОЖНОСТИ
════════════════════════════════════════════════════════════════

1. ПРОСМОТР РАСШИРЕННОЙ ИНФОРМАЦИИ О ТОВАРЕ:
   • Откройте любой товар из списка
   • Прокрутите вниз для просмотра всех новых блоков:
     - Описание
     - Варианты (цвета, размеры)
     - Характеристики
     - Информация о продавце

2. ПРОСМОТР ОТЗЫВОВ:
   • На экране деталей товара
   • Нажмите на блок "Отзывы" (с зеленой иконкой)
   • Откроется экран со всеми отзывами и статистикой

3. ДОБАВЛЕНИЕ НОВЫХ ТОВАРОВ:
   • Все новые данные будут автоматически извлекаться
   • При добавлении товара по артикулу

4. ОБНОВЛЕНИЕ ТОВАРОВ:
   • Нажмите кнопку обновления
   • Все новые данные будут загружены

⚠️ ВАЖНЫЕ ЗАМЕЧАНИЯ
════════════════════════════════════════════════════════════════

1. Не все товары имеют все данные:
   • Некоторые поля могут быть пустыми (null)
   • Интерфейс адаптируется автоматически

   • В этом случае отобразится сообщение "Нет отзывов"
