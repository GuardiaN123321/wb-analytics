════════════════════════════════════════════════════════════════

• Добавление товаров Wildberries по артикулу
• Отслеживание цены, остатков и отзывов
• История изменений с красивыми графиками
• Локальное хранение данных
• Обновление информации о товарах
• Удаление товаров
• Поделиться данными о товаре

🚀 КАК ЗАПУСТИТЬ ПРЯМО СЕЙЧАС:
════════════════════════════════════════════════════════════════

1. Откройте терминал

2. Перейдите в папку проекта:
   cd "/Users/ilgiz/Documents/Аналитика Маркетплейсов/wb_analytics_app"

3. Запустите на macOS (уже доступно!):
   flutter run -d macos

4. Или в браузере Chrome:
   flutter run -d chrome

⚡ ЧТО ДАЛЬШЕ:
════════════════════════════════════════════════════════════════

Для запуска на iPhone:
• Установите Xcode из App Store
• Откройте симулятор: open -a Simulator
• Запустите: flutter run

Для запуска на Android:
• Установите Android Studio
• Создайте виртуальное устройство
• Запустите: flutter run
